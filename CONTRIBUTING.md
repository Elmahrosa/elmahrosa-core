# Contributing

Contributions are accepted under strict constitutional rules.

## Non-Negotiable Constraints
- No contribution may claim or generate sovereign authority
- No governance interpretation or override logic is permitted
- Execution paths must remain auditable and deterministic

## Pull Requests
- Keep changes minimal and scoped
- Document execution impact clearly
- Registry or validation logic requires maintainer approval

## Prohibited
- DAO, token, or voting logic
- Autonomous execution without validation
- Removal of authority references or license notices
